# 🛠️ Como Instalar

### Emulador:
Selecione o jogo e aperte o botão direito,
selecione "Open Mod Data Location" ou qualquer
opção relacionada a abrir pasta de mods.
Jogue a pasta P3P_PTBR_Atlantis de dentro do zip
para dentro da pasta que foi aberta.

### Switch:
No seu Cartão SD COM O ATMOSPHERE, abra a pasta do
atmosphere/contents, crie uma pasta com o nome do
ID do jogo. (Exemplo: 010062B01525C000)
Após isso, jogue as pasta de dentro do P3P_PTBR_Atlantis
(ROMFs e EXEFs) de dentro do zip para dentro da pasta criada. 

Créditos pelo tutorial: Phantasie Translante.
